Политика конфиденциальности

Дата вступления в силу: 24 июня 2026 г.

Разработчик: Андрей Покало

Приложение: OBD Smart Scanner

1. Какие данные собираются

Приложение OBD Smart Scanner собирает только те данные, которые необходимы для его работы:

Данные автомобиля: VIN-номер, DTC-коды ошибок, параметры OBD2 (обороты, температура, скорость, напряжение, нагрузка и другие PID). Эти данные используются исключительно для диагностики и анализа на устройстве пользователя и не передаются третьим лицам, кроме как для обработки запроса к AI-провайдеру (Gigachat или OpenRouter) для анализа результатов диагностики.

Идентификатор устройства: Уникальный ID генерируется при первом запуске для привязки подписки. Не используется для отслеживания.

Данные подписки: При оплате через Telegram Stars передаётся идентификатор заказа и ID пользователя Telegram для подтверждения платежа.

2. Передача данных третьим лицам

Данные диагностики могут передаваться AI-провайдерам (Gigachat, OpenRouter) исключительно для генерации анализа и рекомендаций. Мы не продаём и не передаём данные пользователей рекламным или иным третьим лицам.

3. Хранение данных

Все данные диагностики хранятся локально на устройстве пользователя. На сервере (Cloudflare Worker) хранятся только идентификатор устройства и статус подписки для проверки доступа к премиум-функциям.

4. Безопасность

Передача данных AI-провайдерам осуществляется по HTTPS. Платежи обрабатываются через Telegram Stars — мы не получаем и не храним платёжные данные.
